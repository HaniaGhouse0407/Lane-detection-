## 🧠 Overview

A real-time lane detection pipeline using classical computer vision techniques: Gaussian blur, Canny edge detection, ROI masking, and Hough line transform. All parameters are tunable live through the Streamlit sidebar.

---

## ✨ Features

- ✅ Canny edge detection with tunable low/high thresholds
- ✅ Probabilistic Hough Transform for line detection
- ✅ Region-of-interest masking for road-focused detection
- ✅ Left/right lane separation with slope filtering
- ✅ Adjustable lane line colour, thickness, and overlay opacity
- ✅ Edge map visualisation mode

## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/HaniaGhouse0407/Lane-detection-.git
cd Lane-detection-

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set environment variables (if needed)
cp .env.example .env
# Edit .env with your API keys

# 4. Run
streamlit run app.py
```
